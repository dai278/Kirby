# カービィ夢の泉模倣ゲーム

##本作品は一切の収益化は行っていません。元作品のデータ吸い出しは不使用です。画像は自作ですが一部トレースを含みます。
学習面で制作した物を提示用に共有します。

## プレイ手順（起動〜遊び方）
1. `Game.exe` を起動
2. タイトルで **Z**（決定）
3. ステージを進めてゴールを目指す

## 動作環境
- OS：Windows 10 / 11
- 必要な入力：キーボード
- 画面サイズ：例）1920×1080 推奨
- 注意：未署名exeのため警告が出る場合があります（詳細情報→実行で起動）

## 操作方法（キーボード推奨）
※コントローラー未所持で調整不足のため、キーボードでのプレイを推奨します。

### 通常移動状態
- **A / D**：左右移動（**Shift同時押し**でダッシュ）
- **K**：ジャンプ
- **S**：しゃがみ
- **W**：ホバリング
- **J**：吸い込み
- **ゴール／マップ移動**：ゴールやマップ移動のオブジェクトに触れている状態で、**地面にいる時に W**（決定）

### ホバリング状態
- **W**：上昇
- **Space**：ホバリング解除（空気弾を発射）

### しゃがみ状態
- **S（長押し）**：しゃがみ（押している間だけ）
- **J**：スライディング

### 吸い込み
- **J（長押し）**：吸い込み（敵を吸い込み途中でなければ持続）
- 敵を吸い込むと **頬張り状態** へ移行

### 頬張り状態
- 移動／ジャンプ：通常移動と同じ
- ホバリング：不可
- **J**：吐き出し
- **S**：飲み込み（コピー能力がある敵ならコピー）

### コピー能力中
- 基本操作：通常移動と同じ
- **J**：コピー能力使用
- **Q**：解除

### デバッグ用コマンド
- **1**：死亡
- **2**：体力回復
